# Focus Timer

A minimal Pomodoro-style focus timer that runs entirely in your browser — no install, no account, no ads.

## Features

- Customizable focus and break durations
- Visual progress ring
- Session tracking dots
- Browser notifications when a phase ends
- Audio chime (no external files needed)

## How to use

1. Open `index.html` in any browser
2. Adjust **Focus**, **Break**, and **Sessions** with the +/− buttons
3. Press **Start** — the ring counts down and switches between focus and break automatically
4. After all sessions complete, the timer resets

## Controls

| Button | Action |
|--------|--------|
| Start / Pause | Toggle the timer |
| Reset | Restart the current session from scratch |
| Skip | Jump straight to the next phase |
